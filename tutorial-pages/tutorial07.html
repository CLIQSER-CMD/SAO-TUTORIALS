Top 10 Linux Commands for Beginners

Essential Linux commands every cybersecurity enthusiast should know for navigation, file management, and system analysis.

Why Learn Linux Commands?

Linux is the backbone of cybersecurity. Most security tools, penetration testing distributions, and server environments run on Linux. Understanding basic commands is essential for any cybersecurity professional.

Why Linux Dominates Cybersecurity

  • • Most hacking tools are built for Linux
  • • Servers and cloud infrastructure run on Linux
  • • Open-source nature allows security auditing
  • • Powerful command-line tools for automation
  • • Better control over system processes
  • • Kali Linux and other security distributions

Understanding the Linux Terminal

The terminal is your command-line interface to Linux. It might seem intimidating at first, but it's actually quite powerful and efficient once you learn the basics.

Terminal
user@kali:~$
# This is where you type your commands
# The $ sign indicates you're a regular user
# # sign indicates you're root (administrator)

Terminal Tips

  • • Use Tab to auto-complete commands and filenames
  • • Use ↑ and ↓ arrows to browse command history
  • • Use Ctrl+C to cancel a running command
  • • Use Ctrl+L to clear the terminal screen
  • • Use man command to read manual pages

pwd - Print Working Directory

Shows your current location in the file system. Essential for navigation and understanding where you are.

user@kali:~$ pwd
/home/user/Documents

Why This Matters

When you're deep in directories, it's easy to lose track of where you are. pwd is your GPS in the Linux filesystem.

2
EASY

ls - List Directory Contents

Lists files and directories in your current location. One of the most frequently used commands in Linux.

user@kali:~$ ls
Desktop Documents Downloads Music Pictures Videos
user@kali:~$ ls -la
drwxr-xr-x 2 user user 4096 Nov 27 10:30 .
drwxr-xr-x 3 user user 4096 Nov 27 10:25 ..
-rw-r--r-- 1 user user 220 Nov 27 10:20 .bashrc
-rw-r--r-- 1 user user 807 Nov 27 10:20 .profile

Common ls Options

  • ls - Basic list
  • ls -l - Long format with permissions
  • ls -a - Show hidden files (starting with .)
  • ls -la - Long format with hidden files
  • ls -lh - Human-readable file sizes
3
EASY

cd - Change Directory

Navigate between directories in the Linux filesystem. Essential for moving around and accessing different locations.

user@kali:~$ cd Documents
user@kali:~/Documents$
user@kali:~/Documents$ cd ..
user@kali:~$
user@kali:~/Documents$ cd ~
user@kali:~$

Navigation Shortcuts

  • cd directory - Enter a subdirectory
  • cd .. - Go up one level
  • cd ~ - Go to home directory
  • cd / - Go to root directory
  • cd - - Go to previous directory
4
EASY

mkdir - Make Directory

Create new directories (folders) to organize your files. Essential for maintaining a clean and organized file structure.

user@kali:~$ mkdir cybersecurity
user@kali:~$ mkdir -p projects/2025/january

Useful Options

  • mkdir name - Create single directory
  • mkdir -p path/to/directory - Create nested directories
  • mkdir folder1 folder2 folder3 - Create multiple directories
5
EASY

touch - Create Empty Files

Create empty files or update timestamps. Useful for creating placeholder files or updating file modification times.

user@kali:~$ touch notes.txt
user@kali:~$ touch file1.txt file2.txt file3.txt

Common Uses

  • Create empty files for later use
  • Update file timestamps without modifying content
  • Create multiple files at once
  • Use in scripts to create log files
6
EASY

cat - Display File Contents

View the contents of text files, combine multiple files, or create new files with content. One of the most versatile commands.

user@kali:~$ cat notes.txt
These are my cybersecurity notes
- Learn Linux commands
- Practice with Kali Linux
- Study networking basics
user@kali:~$ cat > newfile.txt
Type your content here
Press Ctrl+D to save and exit

Advanced cat Uses

  • cat file.txt - Display file contents
  • cat file1.txt file2.txt - Combine and display files
  • cat > file.txt - Create file with content
  • cat >> file.txt - Append to existing file
7
MEDIUM

grep - Search Text Patterns

Search for specific text patterns in files. Extremely powerful for finding information in logs, configuration files, or large datasets.

user@kali:~$ grep "password" log.txt
ERROR: Invalid password attempt
Password changed successfully
Failed password for user admin
user@kali:~$ grep -i "error" *.log
system.log: ERROR: Connection timeout
security.log: Unauthorized access error

Useful grep Options

  • grep "pattern" file - Basic search
  • grep -i "pattern" file - Case-insensitive search
  • grep -r "pattern" directory - Recursive search
  • grep -n "pattern" file - Show line numbers
  • grep -v "pattern" file - Show lines NOT matching
8
MEDIUM

ps - Process Status

Display information about running processes. Essential for system monitoring and troubleshooting.

user@kali:~$ ps aux
USER PID %CPU %MEM VSZ RSS TTY STAT START TIME COMMAND
root 1 0.0 0.1 225768 9436 ? Ss 10:20 0:01 /sbin/init
user 1234 0.5 2.3 456789 123456 ? Sl 10:25 0:15 firefox
user 5678 0.1 0.5 234567 34567 ? S 10:30 0:02 terminal
user@kali:~$ ps aux | grep firefox
user 1234 0.5 2.3 456789 123456 ? Sl 10:25 0:15 firefox

Process Information

  • PID: Process ID (unique identifier)
  • %CPU: CPU usage percentage
  • %MEM: Memory usage percentage
  • STAT: Process state (R=running, S=sleeping)
  • TIME: Total CPU time used
9
MEDIUM

kill - Terminate Processes

Stop running processes using their Process ID (PID). Useful for terminating unresponsive programs or managing system resources.

user@kali:~$ kill 1234
user@kali:~$ kill -9 1234

⚠️ Use with Caution

Killing system processes can crash your system. Always double-check the PID and use kill -9 only as a last resort.

Kill Signal Types

  • kill PID - Graceful termination (SIGTERM)
  • kill -9 PID - Force kill (SIGKILL)
  • kill -15 PID - Termination signal
  • killall process_name - Kill by name
10
MEDIUM

sudo - Superuser Do

Execute commands with administrative privileges. Essential for system administration and running security tools that require elevated permissions.

user@kali:~$ sudo apt update
[sudo] password for user:
Hit:1 http://kali.org/kali kali-rolling InRelease
Reading package lists... Done
user@kali:~$ sudo nmap -sS target.com
Starting Nmap scan...

⚠️ Administrative Privileges

Be extremely careful with sudo commands. You can damage your system if you run dangerous commands as root. Always double-check before executing.

When You Need sudo

  • Installing or removing software
  • Modifying system files
  • Running network scanning tools
  • Changing system configurations
  • Accessing protected directories
