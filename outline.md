# SAO Cybersecurity Tutorial Website - Project Outline

## Website Structure
- **index.html**: Main landing page with tutorial grid and navigation
- **tutorial-pages/**: Individual tutorial pages (tutorial01.html - tutorial20.html)
- **main.js**: Interactive components and animations
- **resources/**: AI-generated images and assets

## Design System
- **Colors**: Black background (#0a0a0a), Dark Blue (#1a1a2e), Neon Blue (#00d4ff)
- **Typography**: Inter for body, Orbitron for headings
- **Style**: Modern blog aesthetic with cybersecurity theme
